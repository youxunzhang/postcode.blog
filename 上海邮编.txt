邮政编码在英文中有两种主要的说法：

Postal Code：这是最常见、最通用的说法，在全球范围内被广泛使用。

ZIP Code：这是美国特有的说法，全称是 "Zone Improvement Plan" Code。

它们有什么区别？
Postal Code 是一个更广义的术语，指代任何国家用于邮政目的的代码系统。例如，新加坡的邮政编码是 "Postal Code"。

ZIP Code 专门指美国使用的邮政编码系统。如果你在美国境外，通常不会使用 "ZIP Code" 这个词。

举个例子：

英国的邮政编码：A postal code in the UK is like "SW1A 0AA".

加拿大的邮政编码：A postal code in Canada is like "M5V 2H7".

美国的邮政编码：A ZIP code in the US is like "90210".

所以，如果你不确定对方是哪个国家的人，或者想用一个大家都懂的词，用 "Postal Code" 总是最稳妥的选择。
